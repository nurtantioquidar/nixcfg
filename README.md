# Nix Configuration for MacBook Pro M1

This repository provides a simple, declarative Nix configuration to manage your macOS environment on an M1 MacBook Pro. It focuses on three key areas:

- **Declarative Brew and MAS:** Manage Homebrew packages and Mac App Store applications declaratively.
- **Fish Shell:** Set up and configure the fish shell as your interactive shell.
- **Declarative Font Configuration:** Install and manage fonts declaratively.

## Prerequisites

- **Hardware:** MacBook Pro with M1 chip
- **OS:** macOS Monterey, Big Sur, or later is recommended
